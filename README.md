# 🌙 Moonchunk Store

An open-source, self-hosted Minecraft store platform similar to Tebex and MinestoreCMS. Built with Next.js 15, MongoDB, and TypeScript.

## ✨ Features

- 🛒 **Product Catalog** - Manage unlimited products with images, descriptions, and pricing
- 🏷️ **Categories** - Organize products into customizable categories
- 💳 **Payment Processing** - Integrated payment gateway support (Stripe ready)
- ⚡ **Command Queue System** - Automated command execution via Minecraft plugin
- 🎨 **Admin Panel** - Complete control over your store
- 🚀 **Easy Setup Wizard** - Get started in minutes
- 📦 **Inventory Management** - Track stock levels or set unlimited quantities
- 🔌 **Plugin Integration** - REST API for Minecraft plugin communication
- 🎯 **Guest Checkout** - Allow purchases without registration
- 📊 **Order Management** - Track and process customer orders
